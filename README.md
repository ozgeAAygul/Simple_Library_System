# Simple_Library_System
# Simple Library System (Python)

Bu proje, Python'da OOP (Object-Oriented Programming) yapısını öğrenmek için hazırlanmış basit bir kütüphane yönetim sistemidir.  
Kullanıcı; kitap ekleyebilir, listeleyebilir ve kitap silebilir.

---

##  Özellikler
- Yeni kitap ekleme  
- Kütüphanedeki tüm kitapları listeleme  
- Kitap başlığına göre kitap silme  
- İki sınıf yapısı:
  - `Book`
  - `Library`
- Terminal üzerinden kullanıcı etkileşimi

---

## Kullanılan Yapılar
- Class  
- `__init__()`  
- Metodlar  
- Listeler  
- Döngüler  
- Nesne yönetimi  
- Koşullu yapılar (if-else)
